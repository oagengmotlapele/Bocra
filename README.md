# BOCRA Website

This is a Python-based web application for BOCRA. The app is designed to be responsive, secure, and feature-rich. It includes live charts, user authentication, and email verification.

## Features

- **Live Charts** – Visualize real-time data directly on the dashboard.  
- **Mobile Responsive** – Fully responsive design for desktops, tablets, and mobile devices.  
- **User Authentication** – Register and login functionality with email verification.  
- **Secure Login & Register** – Users must verify their email to access the system.  
- **Admin & User Roles** – Admin can manage content, users can interact with allowed sections.  

## Requirements

- Python 3.x  
- MySQL Database  
- Required Python libraries (install via `pip install -r requirements.txt`)

## Setup

1. Make sure your MySQL database is running and configured correctly.  
2. Update the database settings in your Python code as needed.  
3. Install required Python libraries:
4. download src.zip

```bash
pip install -r requirements.txt
```

```bash
python main.py
```
